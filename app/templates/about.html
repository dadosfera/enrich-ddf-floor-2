About NCM Enricher DDF

A comprehensive tool for working with Brazilian Import/Export product classification data.

What is NCM?

Nomenclatura Comum do Mercosul (NCM) is the classification system used by Brazil and other Mercosur countries for import and export products.

The NCM system is based on the Harmonized System (HS), which is an international nomenclature for the classification of products developed by the World Customs Organization (WCO).

The NCM code consists of 8 digits:

  • The first 6 digits follow the Harmonized System (HS)
  • The last 2 digits are Mercosur-specific subdivisions

For example, in the NCM code 8542.31.00:

  • The first 2 digits (85) represent the chapter
  • The next 2 digits (42) refine the classification within the chapter
  • The next 2 digits (31) further refine the classification
  • The last 2 digits (00) are the Mercosur-specific subdivisions

Structure of NCM Classification

The NCM classification is organized hierarchically:

Sections

The NCM is divided into 21 sections that group similar products.

Chapters

Each section contains multiple chapters (identified by the first 2 digits of the NCM code).

Products

The most detailed level, identified by an 8-digit code.

About This Application

The NCM Enricher DDF application provides easy access to NCM classification data for use in import/export operations, trade analysis, and customs compliance.

Key features:

  • Search NCM products by code or description
  • Browse the NCM classification structure
  • View detailed product information, including tax rates
  • Access data through a RESTful API

Data Source: The NCM data in this application is sourced from official Mercosur and Brazilian government publications.
