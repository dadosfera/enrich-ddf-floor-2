Guia de Integração

1. Inclusão de CSS e JavaScript

<!-- No head do seu layout --> <link rel="stylesheet" href="/static/css/components/loading.css"> <!-- No final do body do seu layout --> <script src="/static/js/components/loading.js"></script>

2. Adicionando Shimmers para Carregamento

<!-- HTML no seu template --> <div id="search-results-container"> <!-- Os resultados serão inseridos aqui quando carregados --> </div> <script> // Quando a página carrega, mostrar o shimmer document.addEventListener('DOMContentLoaded', function() { const container = document.getElementById('search-results-container'); ShimmerPlaceholders.createSearchResults(container); // Simular carregamento (na vida real seria uma chamada AJAX) setTimeout(function() { // Substituir shimmer pelos resultados reais container.innerHTML = 'Resultados carregados!'; // Aplicar animação aos resultados PageTransitions.fadeIn('#search-results-container'); }, 2000); }); </script>

3. Transições Entre Páginas

<!-- No seu layout principal --> <script> // O LoadingManager já é inicializado automaticamente // e intercepta cliques em links e submissões de formulários // Para mostrar o overlay manualmente: // window.loadingManager.showLoading(); // Para esconder o overlay manualmente: // window.loadingManager.hideLoading(); // Para casos especiais onde você não quer o overlay: // Adicione o atributo data-no-loading aos links ou forms // <a href="/pagina" data-no-loading>Link sem overlay</a> </script>
